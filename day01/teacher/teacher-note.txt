浏览器地址栏
Apache闪退(c盘下的Apache24)
	1.80端口占用
		更改C:\Apache24\conf\httpd.conf文件，将80全文替换为8080
	2.443端口占用
		更改C:\Apache24\conf\extra\httpd-ssl.conf和httpd-ahssl.conf。将内部的443替换为442
	3.bin->httpd.exe，双击
	在浏览器器地址栏输入localhost:端口号 回车

本地仓库   本地的文件夹   


想把远程仓库的代码复制下来  
	1.直接下载压缩包
	2.git clone git地址

上传
	git add *       将当前修改的文件添加到上传列表
	git commit -m '刘亚蓉提交'   提交到本地缓存中
		git config --global user.email "you@example.com"
  	git config --global user.name "Your Name"
	git push origin master 将本地缓存中文件推送到远程仓库中

更新
	远程仓库代码发生改变，本地代码需要更新
	git pull origin master
